=== Release directory ===
This directory contains the newest compiled version of the framework. 

It also contains files, that are neccessary to launch the framework: 
	- addons: contain the Dota2ai and MapperScrapper addons
	- configurations: contain some sample scenarios you can use
	- data: contains framework config, map.data and items.data
	- lib: this is a directory where we compile all the dependencies
	- (logs/out): created by the framework for output files

See User Guide or Programmer's Documentation for details.
