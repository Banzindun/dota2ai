DOTA 2 AI FRAMEWORK
================================
This is a framework for developement of inteligent agents for Dota 2.

!For details on how to use it, please see User Guide or Programmer's doc, 
that are located inside the doc folder!

The project contains: 
	- Dota2Framework: source codes of the Dota2ai framework
	- ExampleAgents: source codes of some implemented agents
	- doc: contains javadoc, User Guide and Programmer's documentation.
	- release - contains the newest compiled version of the framework
